# IMG420-Assignment6
Retrieve the Sun is a simple top-down combat game where the player has to defeat all the dark enemies in order to retrieve the stolen sun. The game includes a simple combat system with simple Enemy AI behavior implemented using a Behavior Tree.

# Movement:

WASD to move

SHIFT to sprint

SPACE to attack

# Enhancements:

Added attack and dying animations. Included a particle effects for the player spin attack.

Added sound effects for attack, damage taken, and background music to create a overall more immersive experience.

Enemies and Allies have varying power levels. The enemies are stronger, but more passive as they flee and return to patrolling when the player gets far away enough. The Allies are weaker, but do not stand down and keep attacking. This creates a more balanced combat experience.
